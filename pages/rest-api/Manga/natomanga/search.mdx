# Search

Technical details regarding the usage of the search function for the NatoManga provider can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Route Schema (URL)

```
http:localhost:3000/search?keyw={keyw}&page={page}
```

## Path Parameters

| Parameter |  Type  | Description                                                       | Required? | Default |
| :-------: | :----: | ----------------------------------------------------------------- | :-------: | :-----: |
|   keyw   | string | The search query; i.e. the title of the item you are looking for. |    Yes    |         |
|   page   | string | The search query; i.e. the title of the item you are looking for. |    No    |    1     |

## Request Samples

<Tabs items={["JavaScript", "Python"]}>
  <Tab>
    <>
        ```js
        import axios from "axios";

        // Using the example query "demon".
        const url = "http:localhost:3000/search?keyw=Demon&page=1";
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
    </>

  </Tab>
  <Tab>
    <>
        ```python
        import requests

        # Using the example query "demon".
        url = "http:localhost:3000/search?keyw=Demon&page=1";
        response = requests.get(url)
        data = response.json()

        print(data)
        ```
    </>
