# Get Completed Manga

Technical details regarding the usage of the get manga info function for the NatoManga provider can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Route Schema (URL)

```
http://localhost:3000/manga/natomanga/completed-manga?page=1
```

## Query Parameters

| Parameter |  Type  | Description                                                                                            | Required? | Default |
| :-------: | :----: | ------------------------------------------------------------------------------------------------------ | :-------: | :-----: |
|    page     | string | The NatoManga ID of the manga; i.e. provided by searching for said manga and selecting the correct one. |    No    |    1     |

## Request Samples

<Tabs items={["JavaScript", "Python"]}>
  <Tab>
    <>
        ```js
        import axios from "axios";
        
        const url = "http://localhost:3000/manga/natomanga/completed-manga?page=1";
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
    </>

  </Tab>
  <Tab>
    <>
        ```python
        import requests

        url = "http://localhost:3000/manga/natomanga/completed-manga?page=1"
        response = requests.get(url)
        data = response.json()

        print(data)
        ```
    </>

  </Tab>
