# Get Manga Info

Technical details regarding the usage of the get manga info function for the NatoManga provider can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Route Schema (URL)

```
http://localhost:3000/details/{id}
```

## Query Parameters

| Parameter |  Type  | Description                                                                                            | Required? | Default |
| :-------: | :----: | ------------------------------------------------------------------------------------------------------ | :-------: | :-----: |
|    id     | string | The NatoManga ID of the manga; i.e. provided by searching for said manga and selecting the correct one. |    Yes    |         |

## Request Samples

<Tabs items={["JavaScript", "Python"]}>
  <Tab>
    <>
        ```js
        import axios from "axios";

        // Using the example ID of "259dfd8a-f06a-4825-8fa6-a2dcd7274230".
        const url = "http://localhost:3000/details/super-dragon-ball";
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
    </>

  </Tab>
  <Tab>
    <>
        ```python
        import requests

        # Using the example ID of "259dfd8a-f06a-4825-8fa6-a2dcd7274230".
        url = "http://localhost:3000/details/super-dragon-ball"
        response = requests.get(url)
        data = response.json()

        print(data)
        ```
    </>
