# Get Manga Chapter Pages

Technical details regarding the usage of the get manga chapter pages function for the mangadex provider can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Route Schema (URL)

```
http://localhost:3000/manga/natomanga/chapters/{id}/{chapter_id}
```

## Query Parameters

| Parameter |  Type  | Description                                                                                              | Required? | Default |
| :-------: | :----: | -------------------------------------------------------------------------------------------------------- | :-------: | :-----: |
| id | string | Example: land-of-monsters |    Yes    |         |
| chapter_id | string | Example: chapter-10 |    Yes    |         |

## Request Samples

<Tabs items={["JavaScript", "Python"]}>
  <Tab>
    <>
        ```js
        import axios from "axios";

        // Using the example chapter ID of "chapter-10"
        // Using the example id of "land-of-monsters"
        const url = "http://localhost:3000/manga/natomanga/chapters/land-of-monsters/chapter-10";
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
    </>

  </Tab>
  <Tab>
    <>
        ```python
        import requests

        # Using the example chapter ID of "chapter-10"
        # Using the example id of "land-of-monsters"
        url = "http://localhost:3000/manga/natomanga/chapters/land-of-monsters/chapter-10"
        response = requests.get(url)
        data = response.json()

        print(data)
        ```
    </>
